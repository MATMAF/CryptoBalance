# CryptoBalance
Email notifier when your crypto balance changed.
## Crypto supported
* Bitcoin
* Ethereum
* Dogecoin
* Litecoin
## How to use ?
### Hosted by myself
https://cryptobalance.dev
### Self hosted
```docker
docker compose up -d
```
After 1 minute go to http://127.0.0.1:5000
## Languages and frameworks
* Python
* Flask
* Docker
* MySQL
* HTML
* Tailwind CSS
